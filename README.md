  # 📚 Coaching Institute Website

A modern, responsive website designed for a coaching institute to showcase courses, attract students, and manage student inquiries effectively.


## 📌 Project Overview

This project is a clean and professional website built for a coaching institute. It highlights courses, builds trust with students and parents, and encourages enrollments through clear and persuasive content.

---

## 🎯 Features

* Homepage with strong value proposition
* Course / Services sections (NEET, JEE, Foundation, etc.)
* Contact and Enrollment Call-To-Actions
* Testimonials and trust-building content
* Fully responsive design
* Simple and user-friendly interface

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

## 🌐 Deployment

This project is deployed using GitHub Pages.
